Framework Adapters

Arcis has built-in adapters for the most popular web frameworks. The core functions are framework-agnostic, so Arcis also works with any framework that isn't listed here.

Express (Node.js)

The primary Node.js adapter. Works with Express 4 and 5.

import express from 'express';
import { arcis } from '@arcis/node';

const app = express();
app.use(express.json());
app.use(arcis());

Fastify / Koa / Hono (Node.js)

Built-in adapters are on the roadmap. Today, use the core sanitize functions directly:

// Fastify
import { sanitizeObject } from '@arcis/node';

fastify.addHook('preHandler', async (req, reply) => {
  if (req.body) req.body = sanitizeObject(req.body);
  if (req.query) req.query = sanitizeObject(req.query);
});

FastAPI (Python)

from fastapi import FastAPI
from arcis import ArcisMiddleware

app = FastAPI()
app.add_middleware(ArcisMiddleware)

For custom config:

from arcis import ArcisMiddleware, Config

app.add_middleware(
    ArcisMiddleware,
    config=Config(
        sanitize_xss=True,
        rate_limit_max=60,
    ),
)

Flask (Python)

from flask import Flask
from arcis import Arcis

app = Flask(__name__)
Arcis(app)

Django (Python)

Add to MIDDLEWARE in settings.py:

MIDDLEWARE = [
    'django.middleware.security.SecurityMiddleware',
    'arcis.django.ArcisMiddleware',   # add this line
    'django.middleware.common.CommonMiddleware',
    # ... rest of your middleware
]

Gin (Go)

package main

import (
    "github.com/gin-gonic/gin"
    arcisgin "github.com/GagancM/arcis/gin"
)

func main() {
    r := gin.Default()
    r.Use(arcisgin.Middleware())
    r.Run()
}

Echo (Go)

package main

import (
    "github.com/labstack/echo/v4"
    arcisecho "github.com/GagancM/arcis/echo"
)

func main() {
    e := echo.New()
    e.Use(arcisecho.Middleware())
    e.Start(":8080")
}

net/http (Go)

Use the core Arcis middleware with the standard library:

package main

import (
    "net/http"
    "github.com/GagancM/arcis"
)

func main() {
    mux := http.NewServeMux()
    mux.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        w.Write([]byte("ok"))
    })

    handler := arcis.Wrap(mux)
    http.ListenAndServe(":8080", handler)
}
